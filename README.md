# GitHub Trending CLI

A simple CLI tool to fetch and display trending repositories from GitHub.

## Installation

```bash
npm install
```

## Usage

```bash
npm run build
npm start -- --duration day
npm start -- --duration week
npm start -- --duration month
npm start -- --duration year
npm start -- --limit 5
npm start -- --duration day --limit 5
npm start -- -d day -l 5
```
